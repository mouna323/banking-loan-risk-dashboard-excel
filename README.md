#  Banking Loan Risk Dashboard (Excel)

#  Project Overview
This project presents an **interactive Banking Loan Risk Dashboard** built using Microsoft Excel.
It analyzes customer data to identify **loan distribution, customer risk levels, and default behavior**, supporting data-driven decision-making in a banking context.

#  Objectives
* Analyze customer financial and demographic data
* Identify high-risk customer segments
* Evaluate loan distribution patterns
* Measure and monitor default rates

# Tools & Techniques
* Microsoft Excel
* Pivot Tables & Pivot Charts
* Excel Tables (structured data)
* Advanced Formulas (IF, nested IF, aggregation)
* Data Cleaning & Feature Engineering
* Slicers for interactivity

# Data Preparation
* Preserved raw data and created a separate clean dataset
* Handled missing values using:
  * Median for numerical fields
  * “Unknown” for categorical fields
* Standardized column names for consistency
* Created new analytical features:
  * Income Segment
  * Loan Size Category
  * Risk Level
  * Default Status

# Dashboard Features
* **KPI Cards**
  * Total Customers
  * Average Income
  * Total Loan Amount
  * Default Rate
* **Visualizations**
  * Loan Distribution by Size
  * Customer Risk Segmentation
  * Default Rate Analysis
* **Interactivity**
  * Slicers for:
    * Education Level
    * Risk Level
    * Loan Size Category
## Key Insights
* Majority of loans are large (~80%), indicating concentration in high-value lending
* Extremely high default rate (~88%), highlighting significant portfolio risk
* High-risk customers dominate the dataset
* Customer segmentation reveals strong variation across education and risk levels
